Study:


  1. 프로그래밍 언어 및 기초 프로그래밍

    2. 프로그래밍 언어(C, Python, Java 등)를 배우며 기본적인 프로그래밍 개념
    (변수, 조건문, 반복문 등)을 익히는 과목입니다. 이후 알고리즘 구현,
    문제 해결 능력을 기르기 위해 더 깊이 있는 프로그래밍을 학습하게 됩니다.

  2. 자료구조(Data Structures)

    3. 데이터를 효율적으로 저장하고 관리하는 방식에 대해 배우는 과목입니다.
    배열, 링크드 리스트, 스택, 큐, 트리, 그래프 등의 자료구조를 공부하고,
    이를 통해 프로그램의 성능을 최적화하는 방법을 익힙니다.

  3. 컴퓨터 시스템 구조(Computer Architecture)

    4. 컴퓨터가 실제로 어떻게 동작하는지에 대한 이해를 돕는 과목으로,
    CPU, 메모리, 저장 장치 등 하드웨어 구조에 대해 학습합니다.
    컴퓨터의 내부 동작 원리를 이해하여 시스템 효율성을 높이고,
    소프트웨어와 하드웨어의 상호 작용을 이해하는 데 필수적입니다.

  4. 알고리즘(Algorithms)

    5. 다양한 문제를 해결하기 위한 알고리즘 설계와 분석 방법을 배우는 과목입니다.
    시간 복잡도와 공간 복잡도를 통해 알고리즘의 효율성을 평가하고,
    정렬, 탐색, 동적 프로그래밍 등 여러 알고리즘을 학습합니다.

  5. 운영체제(Operating Systems)

    6. 컴퓨터 하드웨어와 소프트웨어 자원을 관리하는 운영체제의 원리를 학습합니다.
    프로세스 관리, 메모리 관리, 파일 시스템, 입출력 제어 등 운영체제의 주요 기능을 익히고,
    이론적인 개념을 기반으로 실제 운영체제의 작동 방식을 이해합니다.
